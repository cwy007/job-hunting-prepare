# 请列出你所知道的几种 http status code, 并指出他们的意义？

https status code 族繁不及备载，详细内容都可以在网上找到，不过大致分为五类

* 1xx 参考资讯（informational）eg：100 继续
* 2xx 成功（ok）eg：200 确定。客服端请求成功
* 3xx 重新定向 （redirection）eg：301 要求的网页已经永久改变网址。此状态要求客服端未来在连结此网址时应该导向指定的 URI
* 4xx 用户端错误（client error）eg：404 找不到（最常见）
* 5xx 服务器错误（server error）eg：500 内部服务器错误
